# Документація по використанню Dummy Product API

## Загальна інформація
Dummy Product API дозволяє отримувати інформацію про користувачів та товари.
Для доступу до будь-якого ендпоінту потрібен API ключ, який передається в хедері 'X-API-Key'.

## Як отримати інформацію про користувача
Щоб отримати дані про користувача, використовуй інструмент `get_user_info_users__user_id__get`.
Тобі потрібно передати `user_id` як параметр.
Наприклад, щоб знайти користувача з ID 'user123', виклич інструмент з `user_id='user123'`.

## Як отримати інформацію про товар
Щоб отримати дані про товар, використовуй інструмент `get_item_info_items__item_id__get`.
Тобі потрібно передати `item_id` як параметр.
Наприклад, щоб знайти товар з ID 'item_abc', виклич інструмент з `item_id='item_abc'`.

## Важливо про авторизацію
Система автоматично додасть необхідний `X-API-Key` до твого запиту. Тобі не потрібно про це думати, просто викликай інструмент з потрібними параметрами.
